  Image Resizer Tool

  This project resizes and converts images in batch using Python and Pillow.

  Features

- Reads all images from the images/ folder
- Resizes them to 800x800
- Converts them to PNG
- Saves them in the output/ folder

   How to Run

1. Install Pillow:
     pip install pillow

2. Put your images in the images/ folder.

3. Run the script:
     python resize_images.py

4. Resized images will appear in the output/ folder.

   Requirements

- Python
- Pillow (PIL)
